Funcionalidades Gestión de usuarios Registro e inicio de sesión de usuarios. Autenticación y autorización para el acceso a funcionalidades protegidas. Asociación de hechos y reportes a usuarios registrados. Publicación de hechos Creación, edición y eliminación de hechos publicados por los usuarios. Carga de información estructurada (título, descripción, categoría, fecha y ubicación). Asociación de hechos a una posición geográfica para su visualización en el mapa. Soporte para distintos tipos de hechos (eventos, incidentes, reportes ciudadanos, etc.). Reportes Generación de reportes a partir de los hechos cargados. Visualización de reportes consolidados por categoría, zona geográfica o período de tiempo. Exportación o consulta de reportes para análisis posterior. Filtros y búsqueda Filtrado de hechos por: Categoría Rango de fechas Ubicación geográfica Usuario Búsqueda de hechos mediante texto libre. Combinación de múltiples filtros para un análisis más preciso. Visualización en mapa Mapa interactivo con marcadores geolocalizados. Visualización de hechos según filtros aplicados. Agrupamiento de marcadores para mejorar la legibilidad en zonas con alta densidad de datos. Interacción con los marcadores para ver el detalle del hecho. Estadísticas Visualización de estadísticas generales del sistema. Métricas por categoría, ubicación y período de tiempo. Gráficos y resúmenes que permiten identificar patrones y tendencias. Soporte para análisis básico de datos históricos. Arquitectura Arquitectura basada en microservicios. Separación clara entre frontend y backend. Backend expuesto mediante APIs REST. Servicios desplegados en la nube (AWS), preparados para escalar y evolucionar de forma independiente. Seguridad y buenas prácticas Validación de datos en frontend y backend.
